What is Cascading?

The CSS cascade algorithm's job is to select CSS declarations in order to determine the correct values for CSS properties.
When a same element is declared with multiple CSS declarations, then the last declaration(get priorities) will be applied.

What is Specificity?

Specificity is an algorithm that calculates the weight that is applied to a given CSS declaration.
When multiple declarations are applied to the same element, the declaration with the highest specificity will be applied.
If the specificity is the same, then the cascading order will be applied.

The specificity of a CSS selector is calculated based on the following rules:
!important>Inline>Id>Class>Element>Universal

  • Inline styles have the highest specificity. Specificity of 1, 0, 0, 0
  • ID selectors have a higher specificity than class selectors. Specificity of 0, 1, 0, 0
  • Class selectors have a higher specificity than tag selectors. Specificity of 0, 0, 1, 0
  • Element selector have higher specificity than Universal Selector. Specificity of 0, 0, 0, 1
  • Universal selectors have a specificity of 0, 0, 0, 0.
